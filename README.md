# Premium Cinematic Romantic Birthday Website

An Apple-grade, high-performance, cinematic birthday experience created with pure **HTML5**, **CSS3**, and **Vanilla JavaScript**. Zero frameworks, zero external dependencies.

## 🌟 Features
- **Cinematic Workflow**: Smooth scene transitions driven by cubic-bezier curves.
- **Hardware-Accelerated FX Engine**: Integrated canvas particle subsystem supporting real-time stars, twinkle dynamics, and fireworks.
- **Interactive Love Letter**: CSS 3D transforms combined with a custom JS typewriter engine.
- **3D Photo Gallery**: Hardware-accelerated CSS `perspective` carousel.
- **Real-time Counter**: Microsecond-accurate relationship clock.
- **Interactive Candle blowing**: Animated SVG canvas smoke and state execution.
